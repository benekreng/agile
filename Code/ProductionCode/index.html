This is a project created for the UoL CM2020 ASP module as part of the CS BSc.

The following outlines how the system works ...

  1. Stage 1. The first phase is clicking the menu button at the top left
  2. Stage 2. Select what data source will be used for generating graphics by clicking on pre load data button
  3. Stage 3. Upload file
  4. Stage 4. Select relevant files by clicking on the options Button
  5. Stage 5. Generate graph
  6. Stage 6. Choose export type and export graphic
